# CSS Grid Layout Practice

This project contains two different CSS Grid layout examples created to practice and improve CSS Grid skills.

## Features

- CSS Grid Layout
- Grid Template Areas
- Grid Template Columns & Rows
- Responsive grid structure
- Hover effects
- Box shadows
- Card-based dashboard layout
- Custom grid positioning
- Gradient divider styling

## Project Structure

### 1. Dashboard Layout
A dashboard composed of 10 cards using `grid-template-areas`, where each card occupies a specific position inside the grid.

### 2. Grid Areas Exercise
A separate Grid exercise demonstrating how `grid-template-areas` can be used to create complex layouts with eight different elements.

## Technologies

- HTML5
- CSS3
- CSS Grid
